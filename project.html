Project Site Plan

Name

Global NGO Insights

Purpose

The purpose of Global NGO Insights is to inform and educate the public about the valuable work done by non-governmental organizations (NGOs) around the world.

Scenarios

  • Which organizations are active in disaster relief?
  • Which NGOs have active humanitarian projects I can support?
  • What does this NGO actually do, and where do they operate?
  • What are the latest updates on global nonprofit efforts?
  • How are NGOs responding to recent global crises?

Color Schema

  • Secondary - Headings, backgrounds, buttons
  • Gray - Backgrounds, shadows, borders
  • Black - Texts, shadows
  • White - Backgrounds
